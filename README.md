# shortUrl
convert longurl to shortUrl </br>
git clone https://github.com/KWONTAESOO/shortUrl </br>
다운로드 된 폴더로 이동 : 
cd shortUrl </br>
메이븐 인스톨 : 
mvn install </br>
jar 파일로 빌드된 경로 이동 :
cd target </br>
실행 : 
java -jar taez-shorturl-0.0.1-SNAPSHOT.jar </br>

# 실행 후 테스트 방법
ShortUrl 입력 : localhost:9000/web/home </br>
</br>
확인 방법 : localhost:9000/해당 shortUrlKey
